JavaScript

概要

　JavaScriptとは、主にウェブ開発において広く使用されるプログラミング言語であり、インターネットブラウザ上で実行することのできるプログラミング言語である。このJavaScriptの基礎的な活用として、座標やグラフに関する開発をテーマに解説していく。

技術解説

　JavaScriptの基礎的な技術に関して、Pythonのような他の言語の知識も交えて解説していく。

コンソールにログを出力

　console.log()は、JavaScriptのコンソールにメッセージを出力するための関数である。この関数を使用することで、プログラムの途中経過やデバッグのための情報を出力することができる。これはPythonのprint関数や、C言語のprintf関数に近い動作をする。JavaScriptのファイルが実行されているブラウザ上で右クリック＞検証＞Consoleの順に参照することで、動作を確認することができる。

　console.log("Hello, JavaScript!");

変数

　変数はデータを格納することができる箱のようなものであり、変数は宣言をすることで扱えるようになる。 JavaScriptでは、変数を宣言するためにvar、let、constの3つ使用する。 varは再代入可能かつ再宣言可能な変数、letは再代入可能だが再宣言不可能な変数、 constは再代入不可能かつ再宣言不可能な変数を宣言する際に使用する。 変数を宣言する際には、以下のような操作を行う。
　また、JavaScriptは動的型付け言語であるため、変数の型を宣言する必要がない。

　let x = 10; //数値
　const y = 20; //数値
　var z = 30; //数値
　let a = "Hello, JavaScript!"; //文字列

変数に代入されている値が数値である場合、数値の計算を行うことができる。その際には、演算子とよばれる記号を使用する。 以下に、JavaScriptで演算子を使用した数値の計算の例を示す。

　let x = 10;
　let y = 20;
　let z = x + y; //加算
　let a = x - y; //減算
　let b = x * y; //乗算
　let c = x / y; //除算
　let d = x % y; //剰余
